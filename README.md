# FLIQ
A social media photo sharing website built on Django Framework.

The features available to users :-
1) Users can send friend request, cancel requests, accept requests, reject requests or even unfriend their friends.
2) Users can post images with description and tags
3) Users can edit or delete their posts too.
4) Users can edit their profile including profile pic and a short bio about them.
5) Users can check everyone's post and also like and comment on the photos.
6) Without authentication, users can view the posts but cannot like or comment.

Technologies used :-
1) Frontend: HTML5, CSS (Bootstrap4 + Custom CSS), AJAX
2) Backend: Django
3) Storage: SQLite3
